Proyectos

Csg Ecommerce

CSG Ecommerce es una pequeña web desarrollada en PHP que simula una plataforma de compras en línea. Incorpora funcionalidades clave como la gestión de un carrito de compras persistente mediante sesiones,además de personalización del idioma (español/inglés) y la selección de temas (claro/oscuro). Las cookies se utilizan para almacenar preferencias como el idioma y el tema del usuario, garantizando que estas configuraciones se mantengan incluso después de cerrar el navegador. Esto proporciona una experiencia personalizada y consistente para los usuarios..

GitHub
TodoList

To-Do List (lista de tareas) interactiva creada con HTML, CSS y JavaScript, donde los usuarios pueden agregar, eliminar y marcar tareas como completadas. La aplicación utiliza LocalStorage para guardar las tareas de forma persistente, lo que permite que las tareas se conserven incluso después de cerrar o recargar el navegador.

GitHub Demo
COURDEV

API RESTful para gestionar cursos formativos. La aplicación permite a los usuarios visualizar todos los cursos disponibles y filtrarlos por categorías como Desarrollo Web, Programación, Ciberseguridad, Bases de Datos, entre otras. El backend de la aplicación está construído con Node.js y Express, mientras que la base de datos se encuentra alojada de forma remota en Railway y el frontend está desplegado en Vercel..
